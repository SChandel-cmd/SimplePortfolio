Cricket Stats Pipeline GCP

This project details building a cricket statistics pipeline with Google Cloud services. It starts with retrieving data from the Cricbuzz API using Python and storing it in Google Cloud Storage (GCS). A Cloud Function triggers data processing, executing a Dataflow job to transfer data from GCS to BigQuery. The final step involves creating a Looker Studio dashboard for data visualization. This pipeline ensures seamless data flow from retrieval to insightful visualization, showcasing efficient data engineering practices.

Azure: Tokyo Olympics

This project focuses on leveraging Azure services such as Azure Data Factory, Azure Databricks, and Azure Synapse Analytics to process, transform, and analyze Tokyo Olympics data. The data is initially stored in CSV files and uploaded to Azure Storage on Data Lake Gen2. Azure Data Factory orchestrates data movement, Azure Databricks handles data processing and transformation, and Azure Synapse Analytics is utilized for querying and analyzing the transformed data stored in SQL tables.

Kafka: Stock Market Data Streaming

This project aims to simulate a real-time streaming setup for stock market data using AWS services, Kafka, and Python. The workflow involves deploying Kafka on an EC2 instance, generating stock market data in real-time using a Python notebook as a Kafka producer, consuming this data with another Python notebook acting as a Kafka consumer, and storing the processed data in an S3 bucket. Additionally, a crawler is used to automatically populate an Athena table for easy querying.

Youtube Shorts: Reddit Scrapper

This program generates a .mp4 video by querying the top r/askreddit post and collecting comments. It involves installing dependencies, configuring settings, and setting up folders for background media. Users register with Reddit for credentials and update the config.ini file. Modifications enable background music and automatic YouTube uploads using the YouTube Data API. Users configure OAuth credentials in the GCP console and run the script to create and upload videos based on Reddit posts.
